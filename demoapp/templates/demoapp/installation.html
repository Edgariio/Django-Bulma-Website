Django Installation

Step 1: Install latest version of Python at python.org.

Step 2: Open terminal and run "pip install pipenv".

Step 3: Create the folder where your Django project will be with "mkdir project_name". After it is created go into the directory using "cd project_name".

Step 4: When inside the directory, run "pipenv install django". After it is done you should see this message in the image below:



Step 5: Drag and drop the project folder into any Python editor like Visual Studio Code and you are good to go!
